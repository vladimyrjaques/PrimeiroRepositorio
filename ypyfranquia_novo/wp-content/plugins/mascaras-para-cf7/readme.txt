Adicione máscaras de telefone, CPF, CNPJ, CEP e Dinheiro nos campos do Contact Form 7 ou talvez qualquer Input de texto.Adds mask for fields: Phone Number, Zip Code and Money in Contact Form 7 ou text input

== Description ==

Insere uma máscara de telefone no formato Brasileiro ou Americano, tanto com 8 quanto com 9 dígitos.
Formato 8 Digito: (11) 1111-1111
Formato 9 Digito: (11) 11111-1111
Formato Americano: (111) 111-1111
Formato Americano: 111-111-1111

Insere também Máscaras de CPF, CNPJ, CEP e Dinheiro.


Créditos
Foram utilizados os seguintes scripts/serviços de terceiros:
jQuery Mask Plugin (github.com/igorescobar/jQuery-Mask-Plugin).


== Installation ==

A maneira mais rápida de instalar o plugin é:

vá na página: Plugins -> Adicionar Novo
Pesquise por: "mascaras para cf7"
Clique em "Instalar"
Finaliza clicando em "Ativar Plugin & executar Mascaras para CF7"

Se você preferir, você pode instalar de forma manual.

Suba a pasta "mascaras-para-cf7" para a pasta "/wp-content/plugins/"
Ative o plugin dentro do menu "Plugins"

Você encontrará o menu de "Máscaras para CF7" no seu painel de administração do WordPress.

--------------------------------------------------

The quickest method for installing the plugin is:

Go to page: Plugins -> Add New
Search for "mascaras para cf7"
Click in "Install"
Finish clicking in "Activate Plugin & Run Mascaras CF7"

If you prefer, you can install manually.

Upload folder "mascaras-para-cf7" in your FTP folder "/wp-content/plugins/"
Active the plugin in menu "Plugins"

You will find the menu "Máscaras para CF7" in your admin dashboard of WordPress.


== Frequently Asked Questions ==

= Como fazer para a máscara de telefone funcionar no campo? =

Basta usar o campo tipo "tel"

= Como fazer para a máscara de CPF funcionar no campo? =

Basta adicionar a classe cpf ao campo

= Como fazer para a máscara de CNPJ funcionar no campo? =

Basta adicionar a classe cnpj ao campo
